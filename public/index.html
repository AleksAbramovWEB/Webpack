WebPack

  • Инициализация приложения `npm init`
  • Установка webpack гдобально `npm i -g webpack@4.46.0 webpack-cli@3.3.12`, стунут доступны команды webpack через консоль
  • Установка webpack `npm i -D webpack@4.46.0 webpack-cli@3.3.12`
